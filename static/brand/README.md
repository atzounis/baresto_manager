Brand logos are copied from `/images` at the project root:

- `baresto-horizontal.png` ← `images/barestomanager-horizontal-logo.png`
- `baresto-vertical.png` ← `images/barestomanager-vertical-logo.png`

After updating logos in `/images`, recopy:

```bash
cp images/barestomanager-horizontal-logo.png static/brand/baresto-horizontal.png
cp images/barestomanager-vertical-logo.png static/brand/baresto-vertical.png
```
